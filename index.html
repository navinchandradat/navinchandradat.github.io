About

The work behind the numbers

I'm a data analyst based in Toronto with a broad toolkit — R, Python, SQL, Tableau, Power BI, and Excel — and a particular passion for sports analytics. My sports projects are built around a simple principle: frame every finding as something a coach or decision-maker would actually want to know.

On the NFL side, I work with play-by-play data via nflfastR and FTN charting, using EPA, win probability, and coverage metrics to understand how games are won and lost. I'm also expanding into MLB analysis using Statcast pitch-level data through baseballr.

Projects are published as R Markdown reports and deployed here on GitHub Pages — built to be readable by anyone, not just data people.
